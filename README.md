# 코드북 (일러두기)

## 0. 일러두기

이 데이터셋은 여러 연구자가 원문 이미지를 직접 대조해가며 **수작업으로 입력**한 자료입니다. 사료의 특성상 이체자·오식(誤植)·훼손된 지면 등으로 판독이 어려운 경우가 많았고, 여러 차례 교차 검토를 거쳐 신뢰도와 정확도를 높였음에도 불구하고 **오탈자, 누락, 판독 오류가 남아 있을 수 있습니다.**

- 이 데이터셋은 완결된 정본(定本)이 아니라, **지속적으로 보완되는 작업 중인 아카이브**로 봐주시기를 부탁드립니다.
- 연구·인용에 사용하실 때는 가능한 한 원문(신문 지면 이미지)과 대조해 주시고, 확정 여부가 표시된 필드(§4의 "신뢰도 표시 필드" 참고)를 함께 확인해 주시길 권장합니다.
- 오류를 발견하셨다면 §6을 통해 알려주시면 감사하겠습니다.

## 2. 수집 범위와 방법 (요약)

- 대상: 『조선일보』, 『동아일보』 연재소설, 조선일보 278편·동아일보 286편, 총 564편.
- 참고 자료: 한원영, 『한국 신문 연재소설의 사적 연구』 부록 <신문연재소설 연표>; 『신문연재소설전집—원문으로 보는 1930~40년대 소설 모음집』.
- 원문 확인: 네이버 뉴스 라이브러리에 아카이빙된 지면 원문(이미지)을 직접 대조.
- 삽화가 정보는 삽화 하단에 서명이 있거나 삽화가명이 명시적으로 언급된 경우에 한해 확정 표기.
  
## 3. 저작권 관련 처리 방식

**"연재 예고 (기사 제목-링크 대체)", "작가(역자)의 말(기사 제목-링크 대체)", "삽화가의 말"** 세 컬럼은 신문 기사 원문을 그대로 옮긴 것이 아니라, **저작권 보호를 위해 실제 기사 본문 대신 해당 기사의 제목만 기재**한 것입니다.

- 이 컬럼의 값은 논문 본문에 활용된 원문을 그대로 공개한 것이 아니라 **원문을 찾아가기 위한 색인(제목/키워드)**입니다. 본문 전문을 확인하시려면 링크 활용 혹은 "기사 날짜" 필드를 참고하여 네이버 뉴스 라이브러리 등 원문 아카이브에서 직접 확인해 주셔야 합니다.

## 4. 컬럼 설명

### 4-1. 기본 서지 정보

| 컬럼명 | 설명 |
|---|---|
| 일련번호 | 데이터셋 내 고유 일련번호 |
| 연재지면 | 연재 신문 (조선일보 / 동아일보) |
| 작품명 | 원문에 표기된 제목 (한자·구식 표기 포함, 미가공) |
| 작품명(정제) | 현대어로 정제한 제목. '현상당선' 등 현상작 표시, '창작'·'소설' 등 장르 접두어, 작가명 등 부기 정보는 생략하고 통일된 형태로 표기 (자세한 정제 기준은 논문 2장 참고) |
| 작가명 | 원문에 표기된 작가명(필명 포함, 원표기 그대로) |
| 작가명(한글) | 위 작가명의 한글 표기 |
| 작가명(정제) | 조사를 거쳐 통일한 작가 실명 (필명 사용 시 확인된 본명으로 정리) |
| Birth_year / Death_year | 작가 생몰년(확인된 경우에 한함). 미상은 `-` |
| 작가 성별 | M(남) / F(여), 미상은 `-` |

### 4-2. 연재 정보

| 컬럼명 | 설명 |
|---|---|
| 연재시작 / 연재종료 | 연재 시작일·종료일 (YYYY.MM.DD) |
| 연재 관련 정보(확정/추정 코드) | 0 = 시작·종료일 모두 확정, 1 = 시작일 추정, 2 = 종료일 추정, 3 = 시작·종료일 모두 추정. |
| 마지막 회차 | 확인된 마지막 연재 회차 (누락 회차와 무관하게 지면에 표기된 마지막 회차 기준) |
| 분량에 따른 구분 | 단편(1-40회) / 중편(41-99회) / 장편(100회 이상). 기준은 논문 각주 16 참고 |
| 특수 문예 및 특집 구분 | 당선소설, 독자문단, 독자문예, 부인문예, 콩쿨 입선소설, 현상문예 등 특수 기획 표시. 해당 없으면 `-` |
| 장르에 따른 구분 | 역사소설, 영화소설 등 원문에 명시된 장르명. 표시가 없는 경우 `-` (장르 미표시가 곧 무장르를 의미하지는 않음) |
| 비고 | 그 외 특기사항(연작 여부, 확인 가능한 회차 범위, 개제 정보 등) |

### 4-3. 파라텍스트·삽화 정보

| 컬럼명 | 설명 |
|---|---|
| 연재 예고 (기사 제목-링크 대체) | §3 참고. 연재 예고 기사의 제목/색인 |
| 기사 날짜 | 연재 예고 기사가 게재된 날짜 |
| 연재 예고 날짜 | 동일/유사한 연재 예고가 반복 게재된 날짜(복수인 경우 쉼표로 구분) |
| 작가(역자)의 말(기사 제목-링크 대체) | §3 참고. 작가(또는 역자)의 말이 게재된 기사의 제목/색인 |
| 삽화가의 말 | - |
| 삽화가(*추정) | 삽화가명. 지면에 명시적으로 서명/언급된 경우만 확정 표기 |

## 5. 신뢰도 표시 필드

아래 필드들은 "확정 정보"와 "추정 정보"를 구분하기 위한 것입니다.

- `연재 관련 정보(확정(0)/추정(1,2,3))` — 시작·종료일의 확정/추정 여부
- `삽화가(*추정)`의 `*` 표시 — 삽화가 정보의 확정/추정 여부
- 비고란의 "n회부터 확인 가능", "n회차만 확인 가능" 등의 서술 — 결락 회차 존재 여부
